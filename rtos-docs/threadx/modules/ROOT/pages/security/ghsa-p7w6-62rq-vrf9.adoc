== Description

Impact
An attacker can cause arbitrary read and write due to vulnerability in parameter checking mechanism in Azure RTOS ThreadX, which may lead to privilege escalation.

Affected Components
The affected components include RTOS ThreadX v6.2.1 and below. The fixes have been included in ThreadX release 6.3.0.

Remediation
Upgrade to v6.3.0 or above.
