== Description

Impact
An attacker can cause remote code execution due to memory overflow and expired pointer deference vulnerabilities in Azure RTOS NETX Duo.


Affected Components
The affected components include processes/functions related to ftp and sntp in RTOS v6.2.1 and below. The fixes have been included in NetX Duo release 6.3.0.

Remediation
Upgrade to v6.3.0 or above.
