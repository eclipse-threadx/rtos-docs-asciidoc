== Description

Impact
An attacker can cause an out-of-bounds write in Azure RTOS NETX Duo, that could lead to remote code execution.

Affected Components
The affected components include process related to IGMP protocol in RTOS v6.2.1 and below. The fix has been included in NetX Duo release 6.3.0.

Remediation
Upgrade to v6.3.0 or above.
