== Description

Impact
An attacker can cause remote code execution due to expired pointer dereference vulnerabilities in Azure RTOS USBX.

Affected Components
The affected components include components in host class, related to CDC ACM in RTOS v6.2.1 and below. The fixes have been included in USBX release 6.3.0.

Remediation
Upgrade to v6.3.0 or above.
