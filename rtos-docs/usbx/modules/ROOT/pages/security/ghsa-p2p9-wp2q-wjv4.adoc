== Description

Impact
An attacker can cause remote code execution due to memory buffer and pointer vulnerabilities in Azure RTOS USBX.


Affected Components
The affected components include functions/processes in pictbridge and host class, related to PIMA, storage, CDC ACM, ECM, audio, hub in RTOS v6.2.1 and below. The fixes have been included in USBX release 6.3.0.

Remediation
Upgrade to v6.3.0 or above.
