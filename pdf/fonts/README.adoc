= Fonts

This directory contains the Noto Sans and Noto Sans Mono TrueType font files
used when generating PDF manuals.

== Files

`NotoSans-Regular.ttf`, `NotoSans-Bold.ttf`, `NotoSans-Italic.ttf`, `NotoSans-BoldItalic.ttf`::
Noto Sans — body text and headings.

`NotoSansMono-Regular.ttf`, `NotoSansMono-Bold.ttf`::
Noto Sans Mono — code blocks and inline code.
